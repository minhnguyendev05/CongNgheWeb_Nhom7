# Online Course Management System

A PHP & MySQL based online course platform with MVC architecture.

## Features

- **Students**: Browse courses, enroll, track progress
- **Instructors**: Create/manage courses, lessons, materials
- **Admins**: Manage users, categories, approve courses

## Installation

1. Import `onlinecourse.sql` to MySQL
2. Configure database in `config/Database.php`
3. Run on Apache server

## Structure

- Controllers: Handle logic
- Models: Database interactions
- Views: UI templates
- Assets: CSS, JS, uploads

## Usage

Access `index.php` in browser.
